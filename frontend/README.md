# SmartHelp - AI Chat Application

## Description
An AI-powered chat application built using React, Django, and Google Gemini API.

## Features
- Real-time AI chat
- Clean UI with Tailwind CSS
- Chat history support
- PDF export
- Secure backend API

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Django REST Framework
- AI: Google Gemini API

## How to Run
1. Clone repository
2. Setup backend (.env)
3. Start frontend and backend
